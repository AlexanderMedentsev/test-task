# test-task

Установка:
1. ``` docker-compose build ```
2. ``` docker-compose up -d ```
3. ``` docker exec -it <CONTAINER> /bin/sh ```
4. ``` composer install ```

Запуск:
1. ``` ./bin ``` — консольная точка входа в приложение
2. [localhost](http://localhost:80 "Перейти в браузер") — веб точка входа в приложение
